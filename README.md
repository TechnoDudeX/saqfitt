# SAQFITT

Professional coaching and brand partnership site for **SAQFITT** — online fitness coaching for working professionals, plus brand partnership proposals.

## Pages

| Route | File | Purpose |
|-------|------|---------|
| `/` | `index.html` | Main coaching landing page — about, journey, results, process, apply |
| `/links` | `links.html` | Links page (linktree-style) |
| `/media` | `media.html` | Media kit for brand inquiries |
| `/hevy` | `hevy.html` | Hevy Coach partnership proposal |
| `/humehealth` | `humehealth.html` | Hume Health UGC partnership proposal |
| `404` | `404.html` | Custom 404 page |

## Stack

- Single-file static HTML/CSS/JS (no build step)
- Deployed on [Netlify](https://www.netlify.com/) — auto-deploys on push to `main`
- Training signup form handled by Netlify Forms (honeypot spam protection included)

## Local development

Open any `.html` file directly in a browser. No install or build step required.

## Deployment

Push to `main` — Netlify auto-deploys. Form submissions appear in the Netlify Forms dashboard under `training-signup`.
