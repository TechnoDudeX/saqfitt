## Stack

- Single-file static HTML/CSS/JS
- Deployed on [Netlify](https://www.netlify.com/)
- Training signup form handled by Netlify Forms (with honeypot spam protection)

## Setup

1. Clone this repo
2. Connect to Netlify (or drag-and-drop deploy)
3. Enable form notifications in Netlify dashboard: **Site settings > Forms > Form notifications > Email notification**
4. Done

## Form notifications

After deploying, the `training-signup` form will appear in the Netlify Forms dashboard. Add an email notification so submissions get forwarded automatically.

## Local development

Just open `index.html` in a browser. No build step needed.
